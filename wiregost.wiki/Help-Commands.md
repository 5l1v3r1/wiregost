
Help commands show either:
* Command categories
* Commands available in each category
* A specific command

Commands available are the following:

* `help`                    _General Help that shows available command categories and a brief explanation_
* `help {command_set}`      _Show all commands and parameters for this category. Available categories are (core, server, log, chat, db, listeners, stack, exploit, payload)_
* `help {command}`          _Help for a given command_
